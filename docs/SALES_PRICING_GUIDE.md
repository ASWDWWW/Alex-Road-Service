## How to quote (3 steps)

1. **Product mix** — Website only, Platform only, or Full Stack (both).  
2. **Platform tier** — Essentials / Growth / Complete (by modules).  
3. **Customization** — Add billable extras only when the client needs work beyond standard configuration.

**Quote = Build fee + Monthly retainer (+ any customization / add-ons).**

### Website SEO (included with Website only and Full Stack)

Every LaunchPage public website is built so the company can **rank in Google and other search engines** for local and service searches — not just look good as a brochure.

| SEO work included in the build | What sales can say |
|--------------------------------|--------------------|
| Search-friendly page titles, meta descriptions, and headings | “Each page is set up so Google understands who you are and what you do.” |
| Local business signals (NAP consistency, service area, contact details) | “We structure the site around your location and service area.” |
| Clean URLs, mobile-friendly layout, and fast page structure | “Mobile and speed matter for ranking — the site is built for both.” |
| Service and location-oriented page content | “Services and emergency/commercial pages target how customers actually search.” |
| Sitemap / indexing-ready structure at launch | “We launch the site ready to be found and indexed.” |

| SEO ongoing in the website retainer | What sales can say |
|-------------------------------------|--------------------|
| Keep titles, descriptions, and key pages accurate as content changes | “Ranking setup doesn’t go stale when you update services or hours.” |
| Technical health (hosting, SSL, uptime, crawlable pages) | “We keep the site healthy so search engines can keep finding it.” |

**Do say:** The website is **SEO-built for ranking** — structured, local, and search-ready from day one.  
**Don’t say:** Guaranteed #1 rankings, specific keyword positions, or paid Google Ads results (ads are separate and not included).

## Step 2 — Platform feature tiers

Pick the tier that matches the modules the client needs.  
**Full Stack** = Website package + that platform tier (bundle discount already applied).

| Tier | Modules included | Platform only | Full Stack | Best for |
|------|------------------|---------------|------------|----------|
| **Essentials** | Dashboard, Leads*, Customers, Vehicles/Units, Estimates, Work Orders, Invoices, Employees, Settings, role-based access (Admin / Office / Tech), **all-language translation** | **$9,500** build · **$299**/mo | **$12,500** build · **$399**/mo | Small shops leaving paper / spreadsheets |
| **Growth** *(default for most buyers)* | Everything in Essentials **+** Stripe Payments, Inventory, Schedule, Notifications, Media, Reports & CSV exports | **$14,500** build · **$449**/mo | **$17,500** build · **$549**/mo | Roadside, repair, towing, mobile mechanic |
| **Complete** | Everything in Growth **+** Messaging, advanced reporting pack, PM alerts pack, print/export polish, employee onboarding checklist | **$19,500** build · **$549**/mo | **$22,500** build · **$649**/mo | Fleet-heavy / multi-tech shops wanting full RoadReady parity |

\* On **Platform-only** builds, Leads still supports manual entry. Website → inbox automation requires **Full Stack** or adding the Website package later.

### Year-1 total cost of ownership (list, Full Stack)

| Tier | Calculation | Year-1 TCO |
|------|-------------|------------|
| Essentials | $12,500 + 12 × $399 | **~$17,288** |
| Growth | $17,500 + 12 × $549 | **~$24,088** |
| Complete | $22,500 + 12 × $649 | **~$30,288** |

Still well below a typical custom rebuild ($40k–$100k+) and competitive with Fullbay + separate agency website for a multi-user shop.

## Step 3 — What needs customization (pricing)

### Included in every build (do **not** charge extra)

| Item | Note |
|------|------|
| Logo, colors, company copy, services list | Standard brand apply |
| SEO setup for ranking (titles, meta, local/service structure, indexing-ready launch) | Included on Website only and Full Stack builds |
| **All-language website & platform translation** | Globe language switcher on the public site and staff ops platform; translates the full UI into 100+ languages. **Included on every package** (Website only, Platform only, and Full Stack — all tiers) |
| Domain connect or new-domain assistance | Client pays registrar |
| Turn agreed modules on/off | Removing unused modules is free |
| Simple terminology rename (e.g. truck → unit) | Label changes |
| Labor rate, tax, markup, payment terms | Settings configuration |
| Staff accounts + 1 launch training session (≤ 90 min remote) | Included |
| Stripe connection (if Payments selected) | Client owns Stripe account |

### Billable customization (add to **build**)

| Customization | When to charge | List price |
|---------------|----------------|------------|
| Extra website pages | Beyond package page set | **+$250 / page** |
| Custom lead form / qualification flow | Beyond standard Request Service fields | **+$400 – $800** |
| Custom statuses / approval workflow | New estimate or job lifecycle steps | **+$600 – $1,200** |
| Custom field groups (customers / vehicles / jobs) | New data the client must capture | **+$200 / field group** |
| Custom print / PDF layout | Estimate, WO, or invoice redesign | **+$450 / document** |
| Custom report or KPI pack | Beyond Complete reporting | **+$750 – $1,500** |
| Additional staff role (4th+ permission set) | Beyond Admin / Office / Tech | **+$900 / role** |
| Data migration (CSV) | Customers, vehicles, open jobs | **+$1,500 – $4,000** |
| Out-of-scope feature (portal, SMS, QuickBooks, GPS, etc.) | Only if signed into scope | **Custom quote ($2,500+)** |

### Retainer add-ons (monthly or one-time)

## What the monthly retainer covers

### Included

- Hosting & SSL for agreed environments  
- Backups & uptime monitoring  
- Security patches & dependency updates  
- Bug fixes within signed scope  
- **Unlimited staff user seats**  
- Email support (business hours)  
- Minor config tweaks (rates, hours, contact info)  
- **Website packages:** ongoing SEO health (keep pages crawlable; update titles/descriptions when content changes)

### Not included (quote separately)

- New features / modules outside signed scope  
- Major redesigns or rebrands  
- Stripe processing fees  
- Domain registration, **paid search ads (Google Ads)**, managed Google Business posting campaigns  
- Guaranteed keyword rankings or link-building retainers  
- Third-party SMS / accounting / GPS tools  
- After-hours emergency development

## Do not promise without scoped custom work

These are **out of** the current RoadReady reference platform unless added as a custom line:

- Customer portal  
- Customer e-signatures  
- Automatic estimate / invoice email or SMS  
- Payroll / time clock  
- GPS dispatch  
- QuickBooks sync  
- Public appointment booking  
- External calendar sync  
- Purchase ordering  
- Live third-party financing
